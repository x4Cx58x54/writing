数值方法¶

通过推导易得

#include <math.h>
const char *who_is_next(long long n, size_t length, const char *const a[length]) {
    double k = pow(2, ceil(log2(1 + (double)n/length))-1);
    return a[(long long)ceil((n - (k - 1) * length) / k) - 1];
}

模拟方法¶

const char *who_is_next(long long n, size_t length, const char *const a[length]) {
  for (n -= 1; n >= length; n -= length, n /= 2);
  return a[n];
}
